== Installation ==

This section describes how to install the plugin and get it working.


1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

**Useage:**
1.After active this plugin, You will find a "Find me else where" Tab in the sidebar of the dashboard.

2.Confignature the social networks in the dashboard and save.

3.Go to dashboard->appearence->widgets. Drag the find me else where to the sidebar.

4.Enjoy.
